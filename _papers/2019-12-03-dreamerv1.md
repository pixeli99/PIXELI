## 一句话

在 PlaNet 的 RSSM 之上换掉 CEM，改成在 latent imagination 里训 actor-critic，靠对 value 沿 imagined trajectory 的 analytic 梯度回传，在 DeepMind Control 20 个任务上以 5M frames 超过 D4PG 100M frames。

## 方法

- 沿用 PlaNet 的 RSSM 学 world model，再额外学两个网络：actor $q_\phi(a_t\mid s_t)$ 和 value $v_\psi(s_t)$
- imagination horizon $H{=}15$，从 replay 出发 rollout 一条 latent 轨迹
- value target 用 $\lambda$-return $V_\lambda^k$ 平衡 bias/variance：$V_\lambda^k = (1-\lambda)\sum_{n=1}^{H-1}\lambda^{n-1} V_N^n + \lambda^{H-1} V_N^H$
- actor 通过 reparameterization 把 value 梯度直接回传穿过 world model（dynamics 可微）——这是相比 PG 的关键效率来源
- 20 task 平均分 823，PlaNet 332，D4PG 786（用 100M）；样本效率提升约 20×

## 想法

- "把 model-based 写成 actor-critic + imagined rollout"这个 framing 直接把 model-based RL 从 planning 范式拉回 learning 范式，后续 V2 V3 都是在这个框架里加东西
- analytic gradient through dynamics 看起来漂亮，但 horizon 一长就 explode；DreamerV2 的 discrete latent + straight-through 部分就是为了这个
- 没有 image reconstruction loss 的 ablation 显示性能掉很多，说明 representation 主要靠 pixel 重建撑——这个依赖到 V3 才被 symlog 等 trick 缓解

## 一句话

用 **soft slot-based assignment** 替换 hard top-K routing：每个 expert 准备固定数量的 slot，每个 slot 是所有 input token 的加权平均，整条 pipeline fully differentiable，没有 token dropping，没有 aux-loss，在 vision 上同 FLOPs 比 ViT 强一档。

## 方法

- 参数 $\Phi \in \mathbb{R}^{d \times (e \cdot s)}$，$e$ expert 数 $s$ 每 expert slot 数；对 input token 矩阵 $X \in \mathbb{R}^{n \times d}$ 算 $D = \mathrm{Softmax}(X\Phi)$（沿 token 维归一化）
- 每个 slot 的输入是 $\tilde X = D^\top X$，每个 expert 处理自己的 $s$ 个 slot，输出再用 $C = \mathrm{Softmax}(X \Phi^\top)$（沿 slot 维归一化）加权送回每个 token
- 完全没有离散决策，端到端可导；可以理解为"learned attention pooling 进 expert，再 attention 散回 token"
- Soft MoE-Base/16 with 128 expert × 1 slot：JFT-4B 训练 + ImageNet-1k linear probe 顶 ViT-Huge，FLOPs 少 ~5x

## 想法

- 思路漂亮但有个 hard limit：autoregressive LM 没法用。slot 计算需要把当前 batch 的所有 token 一起 softmax，causal mask 一加整个机制就退化。所以这条路线在 vision / encoder-only 里活，LLM 里没人跟
- 与 expert choice 是亲戚（都打破 token-side hard routing），但 soft MoE 更激进：连 sparsity 都没了，每个 expert 看的是所有 token 的 mixture。它已经不算严格意义的 sparse model
- 实际部署价值有限，但作为"routing 不必离散"的 existence proof 很重要。后来 LLM 这边没人捡，可能未来 multimodal encoder 会回来用

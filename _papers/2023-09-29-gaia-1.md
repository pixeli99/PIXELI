## 一句话

Wayve 把驾驶世界建模塑成 next-token prediction：视频 / 文本 / 动作三类输入全部离散化成 token 序列，用一个 9B 的自回归 transformer 预测下一个视觉 token，再过 video diffusion decoder 还原成可看的视频。

## 方法

两段式：

1. **World Model（自回归 transformer，9B 参数）**：图像帧经过 VQ tokenizer 变成离散视觉 token，文本走 CLIP-style encoder，action（速度、曲率）独立 embedding，三类 token 串成一条序列。训练目标是 next-token cross entropy，本质和 LLM 同构。
2. **Video Diffusion Decoder**：把 latent token 序列还原成时序一致的视频帧。

训练数据是 4700 小时的英国城市驾驶视频，由 Wayve 自采。可控性体现在：固定一段历史视频后，给不同 action / text 条件 rollout，能稳定生成"右转 / 直行 / 加速 / 在十字路口刹车"等 counterfactual scenario。

scaling 曲线给得很认真：validation loss 与参数量和 compute 大致符合 power law，作者明显在 frame 这件事："驾驶的世界模型也能像 LLM 那样吃 scale"。

## 想法

GAIA-1 真正的贡献不是某项数字漂亮，而是把"驾驶 world model"从 model-based RL 的 latent dynamics 范式拉回到"生成像素级视频"——这条路后来被 GAIA-2 / Cosmos / Wayve 自己继续推。

但 9B 自回归 + diffusion decoder 的 pipeline 推理慢、可控信号粗（只有速度和曲率两个标量），multi-view、agent box、地图语义这些在 GAIA-2 才被补上。GAIA-1 更像是 viability proof，不是可上车的工具。

## 一句话

11B 参数、训在 30k 小时 2D 平台游戏视频上、**没有任何 action 标注**，却学到了一个 frame-level 可控的 latent action 空间——给一张图 + 用户每帧按一个离散动作（codebook 大小 8），就能 rollout 出可玩的 2D 世界。

## 方法

三组件：
- **Video tokenizer**：ST-ViViT，把视频压成 spatiotemporal token
- **Latent Action Model (LAM)**：从相邻两帧 inverse-dynamics 学一个 VQ codebook 大小 $|\mathcal{A}|{=}8$ 的离散 latent action，VQ 强约束让 codebook 必须紧凑、因此学到的是"控制最有信息量"的轴
- **Dynamics model**：MaskGIT 风格双向 transformer，conditional on $(\mathrm{tokens}_{<t}, a_{t-1})$ 预测下一帧 tokens
- inference 时 LAM 不参与；用户直接按数字 0–7 当 action 喂 dynamics

- 训练数据：30k 小时 internet 2D platformer 视频，过滤后约 244M 16-frame clip
- scaling：4 个尺寸到 11B，loss 单调下降，video FVD / controllability 都随 scale 提升
- 还能用 sketch / photo / text-to-image 生成的图当 prompt 起手

## 想法

- 真正的贡献不是"生成视频"而是"从无标视频里 *涌现出* action vocabulary"——这是 robotics 和 game agent 都垂涎已久的能力，Genie 是第一次大规模证明可行
- codebook 只有 8 这个选择有点 cherry-picked，2D 平台游戏天然 action 少；scale 到 3D / 真实机器人时 codebook 多大、是否分层、连续 vs 离散 全都没回答（Genie 2、V-JEPA 2 AC 才接力）
- 不可控延迟 / 跨长 horizon 漂移 / 物理一致性这些 demo 里其实都看得到，但论文 metrics 选得很温柔
- 这篇是"world model = foundation model"叙事的起点，对后面 Sora / OASIS / Genie 2 整条线都有直接影响

## 一句话

第一个被 DeepMind 明确称为"real-time interactive general-purpose world model"的版本：text prompt 直接出 720p / 24fps 的可玩世界，分钟级一致性。

## 方法

仍是 autoregressive 帧级生成，但显著的工程跳跃在"实时 + 长上下文"两点：

- **720p / 24fps 实时**：在交互过程中边跑边出帧，每一帧都把过去若干秒的画面塞回 context，让 visual memory 保留约 1 分钟内的物体/布局信息。
- **emergent physics**：没有显式 physics engine，物理（碰撞、水、阴影、布料）从大规模 video 训练里涌现，DeepMind 把这一点当成相对 Sora/Veo 的差异化。
- **prompt 类型**：text → world 而不只是 image → world，且世界中途可以被新的 text event 注入（"现在下雨"、"出现一只熊"）。

参数量、训练数据、tokenizer、infra（什么样的卡能跑 24fps）**全部未披露**。仍是 limited research preview，只对部分学者和创作者开放。

## 想法

如果博客 demo 不水分，Genie 3 把 world model 从"研究展示"推到了"具备交互产品体验"的临界点。但 24fps × 720p 不是免费的——多半靠特化推理栈 + 较激进的 distillation，base model 应该比这慢得多。

一分钟一致性仍然是硬上限：embodied agent 训练动辄要几十分钟到数小时的 rollout，Genie 3 这一代仍然在做"长 demo"而不是"长 horizon training data engine"。看哪一代会把"是否能 close loop 喂 RL"作为核心指标披露，那一代才算成。

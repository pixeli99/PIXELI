## 一句话

第一次把 Top-K SAE 那一套机制可解释性工具搬到 diffusion language model 上，发现 DLM 的 SAE feature 比 autoregressive LLM 的更稳定、更好用，做 diffusion-time intervention 比 LLM steering 还强。

## 方法

backbone 是公开 DLM（mask-predict / discrete diffusion 那一类）。在每个 transformer block 后插一个 Top-K SAE：

$$
z = \mathrm{TopK}\bigl(W_\text{enc}(h - b_\text{pre})\bigr), \quad \hat h = W_\text{dec} z + b_\text{pre}
$$

每层独立训练，loss 是 reconstruction + 稀疏度（Top-K 直接卡住活跃数）。

三个发现：

- **早期层插 SAE 反而降 CE loss**：autoregressive LLM 里插 SAE 会有一定的 reconstruction tax，DLM 里没有，甚至在前几个 block 会让模型 perplexity 更好。猜测是 SAE 起到了降噪的作用
- **diffusion-time intervention**：在某个 step 把某个 SAE feature 强制 clamp 住，能够操控生成内容，效果比 activation steering 更可控
- **decoding order signal**：SAE feature 活跃度能预测哪些位置该先 decode，给 DLM 的 unmasking schedule 提供启发

post-training 阶段 feature 基本稳定，说明这些 feature 是预训练就形成的而不是 fine-tune 的产物。

## 想法

DLM 的 mechanistic interpretability 是个空白区——大家研究 autoregressive LLM 的 SAE 已经一年多了，diffusion 这边几乎没人动。这篇就是把 method 平移过来跑通，但"早期层插 SAE 反而降 loss"这个观察是非平凡的，值得跟一波。如果属实，说明 DLM 的早期表征里有大量冗余被 SAE 的稀疏约束顺手 denoise 掉。

不足：没给具体 perplexity 数字（abstract 没写，正文也比较粗），feature 的语义可解释性也没有像 Anthropic monosemanticity 那样做 human-labeling。"clamp feature 做 intervention 比 steering 强"这种比较需要明确的 metric，paper 给的是 qualitative。

diffusion-time intervention 这个 angle 我比较看好——DLM 每个 step 都重新过一遍模型，给了 mechanistic 工具一个"反复施加 patch"的天然窗口，比 AR LLM 的 single-shot decoding 操作面大得多。

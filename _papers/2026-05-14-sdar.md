## 一句话

在多轮 LLM agent 的 RL 训练里，把 on-policy self-distillation 作为一条 token 级 gated 辅助 loss 挂在 GRPO 上，去补 trajectory-level reward 太粗的问题。

## 方法

baseline 是 GRPO：每条 trajectory 只拿到一个 scalar reward，token 级信号几乎是噪声。SDAR 的做法：

- 每个 step 让当前 policy 自己扮演 teacher，对 student 输出打一个 token 级的"是否值得学"的 sigmoid gate $g_t \in [0,1]$
- gate 高的 token 走 distillation loss（拉近 student 与 teacher 分布），gate 低的（teacher 自己都拒绝的）软衰减、不强行复制
- 总目标：

$$
\mathcal{L} = \mathcal{L}_\text{GRPO} + \lambda \sum_t g_t \cdot \mathrm{KL}(\pi_\text{teacher} \,\|\, \pi_\text{student})
$$

RL 仍是主干，distillation 只是补充 token 级监督。在 Qwen2.5 / Qwen3 两个家族上跑 ALFWorld、Search-QA、WebShop：

- ALFWorld: +9.4% over GRPO
- Search-QA: +7.0%
- WebShop: +10.2%

也跑了 hybrid RL+OPSD 的 baseline，SDAR 在所有 model scale 上都稳赢。

## 想法

agent RL 的痛点确实是 sparse credit assignment——给一整条 50 步的 tool-use trajectory 一个 0/1 reward，几乎没法学。SDAR 的 gate 看起来像个软的 token-level advantage 替代，思路比 RLOO / per-step reward shaping 干净。

不放心的点：teacher 就是自己 EMA / 自己 frozen 版本，self-distillation 在数学上等价于一个正则化项，提升的"知识"上限在哪不清楚——尤其 ALFWorld 这种容易 reward hacking 的环境，+9.4 可能一半来自 KL 正则的稳态化。希望看到对 teacher snapshot 频率的 ablation 和 long-horizon（>30 步）任务上的表现。

跟 RLOO、DPO-on-trajectories 的对比也想看，paper 里这块给得不够。

World Models 全景

广义"world model"指一个能在某种状态空间里预测下一步会发生什么的模型——预测目标可以是嵌入（latent）、token 序列，或者直接是像素。"预测得到什么"决定了这套模型能干什么：在 latent 里预测得到的是 sample-efficient 的 RL/planning 引擎；预测像素得到的是 video generation；带动作输入得到的是可玩的虚拟世界。这页把 2018 到 2026 这一支的主要模型放在同一张图上比对。

数据基础：本知识库 _papers/ 中 29 篇笔记 + 6 篇公开但未单独归档的闭源旗舰。

三大流派 — 一句话差异
Latentpredict embedding
RSSM / JEPA 路线——在压缩后的低维状态里预测。不渲染像素，省算力、容易给下游 policy/controller 用，但没法直接给人看视频。代表：DreamerV3、V-JEPA 2。
Tokenpredict next token
把视频/动作 VQ-VAE 量化成 token，套 transformer 做 next-token prediction。可解码回像素也可控、训练栈跟 LLM 同构，但分辨率受 codebook 上限。代表：IRIS、Genie、GAIA-1。
Pixelpredict frame
DiT / latent diffusion 直接生成 RGB 帧序列。视觉天花板最高，能 prompt-conditioned，但推理慢、长 horizon 一致性差，更像电影机不像 simulator。代表：Sora、Veo、Wan、Cosmos。

Benchmark 横切

评测体系正在大迁移。FVD/IS/CLIPSIM 实际上被弃用——Beyond FVD (ICLR 2025) 列了三条致命缺陷：I3D 特征非高斯、对时间扰动不敏感、需要不切实际的样本量。当下分四套接管：VBench-2.0（intrinsic faithfulness，物理/常识/合成性/解剖）、WorldArena（embodied utility，清华 FIB，CVPR 2026 Challenge）、WorldMark（interactive WM 专用，500 case）、WorldScore（Stanford）。

业界 snapshot · 2026 Q2

四大阵营分化已经成型。① 视频生成派（Sora 2 / Veo 3.1 / Wan 2.6 / Kling 3.0 / Seedance 2.0）追"看上去像物理"；② 实时交互派（Genie 3 / Decart Lucy <30ms / WHAMM）追"能玩、能 close loop"；③ 行业派（Wayve GAIA-3、NVIDIA Cosmos、1X、World Labs Marble、Tencent HY-World 2.0）只服务自己业务指标；④ JEPA 派（Meta V-JEPA 2-AC）拒绝预测像素，押注 embedding 空间。四派短期不会合流——评测尺度根本不一样。

开源比例反常地高。foundation WM 这条线开源占 ~60-65%（Cosmos、V-JEPA 2、HY-World 2.0、Wan 2.2、WHAM 全开），real-time interactive 反过来仅 ~30% 开源（Genie 3 / Lucy / Oasis / GAIA-3 全闭）。整个 WM 空间开源比 ~55-60%，与 LLM 顶端 ~70% 闭源的格局完全反转。

"长一致性"仍是公认硬上限。Genie 3 的分钟级是当前 SOTA，但所有人都知道这远远不够 RL 训练所需的小时级 rollout。WorldMark 的 state-consistency 维度 Genie 3 拿 6.4/10（最强），但其他模型普遍 2-6 分——长 horizon 这条线整个 field 都没解决。

资本侧 2026 Q1 同步爆发。World Labs 2026-02-18 完成 $1B / 估值 ~$5B（Autodesk $200M 领投）；Wayve 2026-02-24 完成 Series D $1.26B / 估值 $8.6B；Decart 2026-05-18 完成 $300M Series C / 估值 ~$4B。三家路线分化明显（内容生成 / 驾驶评估 / 实时交互），但都被 NVIDIA Cosmos 收编为 synthetic data 引擎。

Sora 2 已退出舞台。OpenAI app 于 2026-04-26 关闭、API 计划 2026-09-24 停服，用户迁移到 GPT-5 视频功能；这条线 OpenAI 实际放弃了独立产品形态，转入多模态 GPT 主线。

研究空白 — taxonomy 矩阵的三个空格：① latent × content generation（没人用纯 latent 做面向消费者的内容生成，渲染瓶颈）；② pixel × RL/planning（pixel 推理太慢，Cosmos 一次 action 要 240 秒）；③ latent × evaluation（用 embedding 距离做模型能力评估几乎没人在做）。这三块都是真实未开发地。

研究空白 — 当前 field 的三个未开发格

把 30+ 模型塞进 [Latent / Token / Pixel] × [RL planning / Content gen / Evaluation] 的 2×3 矩阵后，三个格子明显空着——这本身就是研究信号：

空格为什么空谁可能填上
Latent × Content Gen latent 不易渲染回像素 → 消费者看不到 有人把 JEPA 嵌入 + diffusion decoder 配对（V-JEPA → frame decoder）
Pixel × RL / Planning pixel 推理慢，Cosmos 一次 action 要 240 秒 Cosmos / SANA-WM 等开始投入；蒸馏 + 量化能压下来吗
Latent × Evaluation 用 embedding 距离做模型能力评估几乎没人在做 V-JEPA 2 的 physical reasoning bench 是孤本——Meta 内部线索

Latent
JEPA / RSSM		 样本效率高（DreamerV3 在 Atari 100k 上击败 model-free）；可直接给 controller / policy 用；不预测像素 = 不浪费算力在不可预测内容上（LeCun 论点） 无法给人看；表达力受 latent dim 上限；评测困难（没有视觉 ground truth）；社区注意力远低于像素派 RL planning · embodied reasoning
Token

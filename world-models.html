World Models 全景 · 2026 Q2

39 个模型 · 4 类评测 · 2018 — 2026 · 数据 2026-05-20
三句话看完 ① World Model = 能预测"下一步发生什么"的模型，2024 年 Sora + Genie 让它从 RL 工具破圈成行业基座。 ② 2025 年起分流为四派：latent / token / pixel / spatial，评测和落地路径都不同，短期不会合流。 ③ 视觉好 ≠ 一致性好，闭源旗舰（Sora 2、Genie 3、Marble、GAIA-3）不在公开 benchmark 榜上；长 horizon 一致性整个 field 都没解决。

§ 1 这是什么定义 · 应用 · 为什么 2024 后才热

广义 World Model 是一个能预测下一步会发生什么的模型。给一段视频 + 一个动作 + 一些条件，它告诉你接下来若干帧会长什么样、嵌入会怎么变、3D 场景会怎么演化。

预测目标的不同决定了它能干什么：

  • 预测嵌入（latent） → sample-efficient 的 RL planning 引擎，给 controller / policy 用。代表 DreamerV3、V-JEPA 2。
  • 预测 token（VQ 离散化） → 可解码、可控、与 LLM 同构的训练栈。代表 IRIS、Genie、GAIA-1。
  • 预测像素（DiT / diffusion） → video generation，视觉天花板最高。代表 Sora、Veo、Wan。
  • 预测 3D 资产 → spatial intelligence，输出可导出的 mesh / gaussian splat。代表 Marble、HY-World 2、VGGT-Ω。

这事儿过去十年都在做（RL 里叫 model-based learning），2024 年突然破圈的两个触发点：① Sora 把"video gen = world simulator"砸进主流叙事；② Genie 的 latent action 让 world model 从"实验室 demo"变成"可玩"。从此机器人公司、自动驾驶公司、游戏公司、空间计算公司同时把它当作底层基座来投。

§ 2 发展三段2018 — 2026

2018 — 2022

起源：RL 圈内的工具，没破圈

起点是 Ha & Schmidhuber (2018) 的 "World Models" 论文——用 VAE 把像素压成 32 维 latent，让 agent 先学世界、再在"梦里"训策略。架构小、思路新，但只 work 在 CarRacing 这类玩具任务上。

这条线被 Hafner 的 Dreamer 系列接住——PlaNet (2018) → DreamerV1 (2019) → V2 (2020) → V3 (2023)——一直在 RL benchmark 上推进。DreamerV3 用单套超参 + 单套架构在 150+ 任务上 work，Minecraft 从零挖钻石，2025 年发 Nature。这是 latent 派至今最强的代表。

并行的是 IRIS (2022) 和 TWM (2023)：把 world model 写成 token 序列上的 GPT-style LM，跟 LLM 工程栈同构——但仍然只在 Atari 这种 RL benchmark 上证明自己。

这一阶段 world model 是 RL 圈内的工具，主流社区不感兴趣。

2023 — 2024

引爆：三件大事砸进主流

① I-JEPA (2023.01, Meta LeCun) 给 latent 派一个哲学口号——"预测像素是徒劳的，应该在 representation 空间预测"。这个口号本身重要，不论是否最终正确。

② Sora (2024.02, OpenAI) 的技术报告标题就叫 "Video generation models as world simulators"。DiT 在 spacetime patches 上 scale，长视频里似乎涌现物理（咬痕、笔触留痕）。这件事让"world model"在大众媒体里第一次成为热词。

③ Genie 1 (2024.02, DeepMind) 从无标注游戏视频里学出 latent action，给用户每帧一个动作，可玩一个 2D 世界。这是第一次让 world model 跑成"游戏"。后续 Genie 2 (2024.12) 出 3D，Genie 3 (2025.08) 做到 720p / 24fps 实时。

同期：Lumiere、VideoPoet、V-JEPA、GAIA-1、Oasis 各自在视频生成 / JEPA / 驾驶 / 实时游戏方向冒出。

这一阶段 world model 从 RL 工具变成行业叙事。VC 开始投钱。

2025 — 2026

分流：四派各自做实，不再装作要造"通用模拟器"

到 2025 年大家意识到：这不是一个问题，是四个问题。视频生成派、可交互派、行业派（驾驶/机器人）、JEPA 派的目标和评测尺度根本不一样。

视频生成派：Sora 2、Veo 3.1、Wan 2.6、Kling 3.0、Seedance 2.0 追"看上去像物理"。开源的 Wan / HunyuanVideo 把视觉差距压到 5 个百分点以内。

可交互派：Genie 3 做到分钟级一致性，Decart Lucy 把延迟压到 30ms 以下，WHAMM 跑 Quake II 实时。"能玩、能 close loop"是他们追的硬目标。

行业派：Wayve GAIA-3 (15B, 2025.12) 明确定位"驾驶安全评估"而非"控制器"；NVIDIA Cosmos 做 Physical AI foundation 平台，给 1X、Figure、Agility 当 synthetic data 引擎；1X 把 world model 嵌进 NEO 人形机器人当认知核心；World Labs Marble 把它做成可下载 3D 资产。

JEPA 派：Meta V-JEPA 2 (2025.06) + 1 小时机器人数据，zero-shot 在 Franka 上完成 pick-and-place 65%。每个 action 推理 16 秒，是 Cosmos Policy 的 1/15——这是 latent 路线最硬的兑现。

评测体系同步换血：传统 FVD/IS/CLIPSIM 被弃用，WorldArena（清华 FIB，CVPR 2026 Challenge）、WorldMark、VBench-2.0、Stanford WorldScore 接管。

四派一目了然

Latent
predict embedding
RSSM / JEPA 路线，压缩态里预测。省算力、给 controller 直接用，但渲染不回像素。代表 DreamerV3、V-JEPA 2。
Token
predict next token
VQ + AR transformer。栈与 LLM 同构，天然支持 action token，但 codebook 决定细节上限。代表 IRIS、Genie、GAIA-1。
Pixel
predict frame
DiT / latent diffusion 出 RGB 帧。视觉天花板最高，但推理慢、长 horizon 漂移、闭源旗舰为主。代表 Sora、Veo、Wan、Cosmos。
Spatial
predict 3D scene
直出可导出 3D 资产（gaussian splat / mesh）。空间一致性强，但动态支持弱。代表 Marble、HY-World 2、VGGT-Ω。

读完这 8 个的判断 ① latent / token / pixel 三派的训练目标天然不同：latent 派的 loss 是 embedding 距离 + KL，token 派是 cross-entropy，pixel 派是 diffusion v-prediction。短期不会合流的本质原因。 ② "无 action 标注学动作"（Genie 的 LAM）和"无像素重建学世界"（JEPA）是过去两年最关键的两个 trick——它们绕开了数据标注的瓶颈。 ③ Cosmos 这种"平台思路"跟单点 SOTA 模型逻辑完全不一样：它在赌 Physical AI 的整条管线，而不是单一指标。

§ 4 当前阵营谁在做什么 · 资本动向

小结 视频生成派和实时交互派的开源/闭源比例完全反转：foundation WM 开源占 60-65%，real-time interactive 仅 30% 开源。LLM 那种"前沿全闭源"在 WM 这条线不成立。

四大阵营分化已成型。 ① 视频生成派（Sora 2 / Veo 3.1 / Wan 2.6 / Kling 3.0 / Seedance 2.0）追"看上去像物理"； ② 实时交互派（Genie 3 / Decart Lucy <30ms / WHAMM）追"能玩、能 close loop"； ③ 行业派（Wayve GAIA-3、NVIDIA Cosmos、1X、World Labs Marble、Tencent HY-World 2.0）只服务自己业务指标； ④ JEPA 派（Meta V-JEPA 2-AC）拒绝预测像素，押注 embedding 空间。四派短期不会合流——评测尺度不一样。

开源比例反常地高。Foundation WM 这条线开源占 ~60-65%（Cosmos、V-JEPA 2、HY-World 2.0、Wan 2.2、WHAM 全开），real-time interactive 反过来仅 ~30% 开源（Genie 3 / Lucy / Oasis / GAIA-3 全闭）。整个 WM 空间开源 ~55-60%，与 LLM 顶端 ~70% 闭源完全反转。

资本侧 2026 Q1 同步爆发。World Labs 2026-02-18 完成 $1B / 估值 ~$5B（Autodesk $200M 领投）；Wayve 2026-02-24 Series D $1.26B / 估值 $8.6B；Decart 2026-05-18 $300M Series C / 估值 ~$4B。路线分化明显，但都被 NVIDIA Cosmos 收编为 synthetic data 引擎。

Sora 2 已退出舞台。OpenAI app 2026-04-26 关闭、API 2026-09-24 停服，用户迁移到 GPT-5 视频功能；OpenAI 实际放弃了独立产品形态。

§ 5 评测怎么看4 张主榜

结论 传统 FVD / IS / CLIPSIM 已被弃用。当下并存 4 套：① WorldMark（interactive WM）② VBench-2.0（视频物理常识）③ 机器人成功率（具体任务）④ AA Elo（人工偏好）。视觉好 ≠ 一致性好，闭源旗舰大都缺席统一榜单。

§ 7 还没解决研究空白 + 长 horizon

把 39 个模型塞进 [Latent / Token / Pixel] × [RL planning / Content gen / Evaluation] 的 2×3 矩阵后，三格明显空着——这本身就是 field 的研究信号。
空格为什么空谁可能填上
Latent × Content Generation latent 不易渲染回像素，消费者看不到 JEPA 嵌入 + diffusion decoder 配对（V-JEPA → frame decoder）
Pixel × RL / Planning pixel 推理太慢，Cosmos 一次 action 240 秒 Cosmos / SANA-WM 已投入；蒸馏 + 量化能压下来吗
Latent × Evaluation embedding 距离做模型能力评估几乎没人在做 V-JEPA 2 的 physical reasoning bench 是孤本
最硬的天花板 长 horizon 一致性。Genie 3 的"分钟级"是 SOTA，但 RL 训练需要小时级 rollout。WorldMark state-consistency Genie 3 拿 6.4/10，其他模型 2-6 分。整个 field 都没解决——这是下一代要攻的核心。

§ 8 一句话总结汇报时讲这五条

  1. World Model 已破圈。从 RL 工具（2018-2022）→ 行业基座（2024 起），背后是 Sora + Genie 的两次"破墙"时刻。
  2. 四派分流，短期不会合流。视频生成 / 实时交互 / 行业（驾驶 + 机器人 + 空间）/ JEPA，评测尺度根本不一样。
  3. 当下 SOTA 没有统一答案。视觉看 Wan 2.6 / Veo 3.1；交互看 Genie 3；驾驶看 GAIA-3；机器人看 V-JEPA 2-AC + Cosmos Policy；3D 看 Marble + HY-World 2。
  4. 开源比 LLM 反常地强。foundation WM 60-65% 开源；real-time interactive 闭源主导（Genie 3 / Lucy / GAIA-3）。
  5. 未解的核心是长 horizon 一致性。分钟级到小时级还差几个量级，谁先解决谁能把 WM 变成真正的 close-loop 训练引擎。
